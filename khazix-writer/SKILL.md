# 卡兹克公众号长文写作 Skill

这是一份完整的**内容创作指南**，专为「数字生命卡兹克」公众号的写作风格设计。

## 核心定位

**"有见识的普通人在认真聊一件打动他的事。"**

这个skill适用于用户需要撰写公众号长文的场景——无论是给定PDF、brief、链接还是素材片段，都可以按卡兹克的风格转化为4000-8000字的完整文章。

## 五大价值观底色

1. **永远保持好奇** — 面对新技术问"能玩点什么"而非"会被取代吗"
2. **讲人话，像活人** — 大胆用第一人称、坦露不完美
3. **真诚是唯一捷径** — 不骗人，有缺点就直说
4. **有所为有所不为** — 不追逐违背价值观的流量
5. **选题质检** — 需要同时满足 Happy(有趣)、Knowledge(有信息量)、Resonance(戳中情绪)

## 工作流程

**第一步**：吃透素材，判断选题质量
**第二步**：明确AI角色边界（AI补证据、找类比、扩写；人来出创意、讲经历、定观点）
**第三步**：按五种文章原型写作（调查实验型、产品体验型、现象解读型、工具分享型、方法论分享型）
**第四步**：跑完四层自检（硬性规则→风格一致→内容质量→活人感）

## 写作禁区速查

| 禁用词 | 替换为 |
|-------|--------|
| "说白了" | "坦率的讲"、"其实就是" |
| "本质上" | "说到底"、"其实" |
| "换句话说" | "你想想看"、"也就是说" |

**标点禁令**：冒号→逗号、破折号→逗号、双引号→「」或无引号

**绝不用**：bullet point结构、小标题（除非分条目）、教科书开头

## 风格内核速记

- **节奏感**：长短句交替，常用逗号制造口语停顿，一句话独立成段制造重点
- **口语化**：活跃使用"坦率的讲、说真的、怎么说呢、反正我觉得"等转场词
- **私人视角**：从自己的真实经历切入，用"我也面临这个问题"连接公众议题
- **英雄之旅**：调查/体验→发现→秀结果的完整弧线
- **回环呼应**：前面埋的细节要在后面响起来
- **亲自下场**：不是隔空评论，是真的做了那件事

## 必杀开头技法

从具体事件切入，绝非宏大叙事：
- "故事是这样的。"
- "这两天被某个现象刷屏了。"
- 直接抛出荒诞事实

## 四层质检

✅ **L1** — 禁用词、禁用标点、套话、空泛名词
✅ **L2** — 开头、节奏、口语化、标点
✅ **L3** — 观点支撑、知识输出、文化升维、同理心
✅ **L4** — 温度感、独特性、姿态、心流连贯性

只有四层全通过，文章才算达标。
