# 公众号爆款文章评估框架

> 7 DIMENSIONS · KEY METRICS · BENCHMARK STANDARDS
>
> 核心链路：**标题拉打开 → 钩子拉完读 → 密度拉收藏 → 情绪拉转发 → 引导拉评论 → 人设拉关注 → 裂变拉二次传播**

---

## 七个维度 · 评分标准

### D1 · 标题与封面（打开率 ≥ 10%）

**核心公式**：信息差 + 情绪触发 + 具体数字（命中 ≥ 2/3）

- 悬念、反常识、身份共鸣 — 至少命中其中一个
- 封面图在信息流中形成**视觉中断**，避免通用素材图
- 关键词前置（前 10 字决定是否停下来）
- 字数 ≤ 25 字，手机端不截断

**检查清单**：
- [ ] 标题含有具体数字 或 反常识结论 或 强身份共鸣
- [ ] 封面图非通用素材，有视觉差异化元素
- [ ] 标题承诺与正文内容一致（不过度承诺）

---

### D2 · 钩子结构（完读率 ≥ 30%）

**核心公式**：前 3 行给出"这篇和你有关"的信号

- 数据冲击 / 场景痛点 / 反直觉结论 — 前置，不铺垫
- 用户划走只需 0.5 秒，绝不能用套话开场
- Hook 类型：场景代入 / 反常识 / 数字冲击 / 问题悬念

**检查清单**：
- [ ] 前 3 行没有"在当今...""随着...""我们需要..."
- [ ] 前 3 行出现了具体数字 或 场景描述 或 反常识结论
- [ ] 读完第一段有继续读下去的欲望

---

### D3 · 内容密度与节奏（收藏率 ≥ 3%）

**核心公式**：每屏都有"读下去的理由" + 工具价值

- 段落 2-4 行以内，手机屏幕可见范围
- 善用**加粗**、编号、金句卡片做视觉分层
- 收藏 = 内容有工具价值，用户认为值得回看

**检查清单**：
- [ ] 段落控制在 2-4 行，无超过 6 行的大段
- [ ] 每 300 字有 1-2 个加粗关键词
- [ ] 至少有 1 个"可回收的干货"（清单/对比表/操作步骤）

---

### D4 · 情绪设计（分享率 ≥ 5%）

**核心公式**：让读者转发时完成一次自我表达

- 转发 = 社交货币：彰显品位 / 产生共鸣 / 利他分享
- 愤怒、焦虑、自豪、感动 — 四类情绪传播力最强
- 文章要有情感曲线：紧张-释放的节奏交替

**检查清单**：
- [ ] 全文至少触发一种强传播情绪（愤怒/焦虑/自豪/感动）
- [ ] 有"读者转发时想表达什么"的明确设计
- [ ] 情感不是一马平川，有起伏节奏

---

### D5 · 互动引导（评论率 ≥ 1%）

**核心公式**：埋问题 + 结尾显式引导

- 文中埋开放式问题 或 争议性判断，引发表达欲
- 结尾用投票/提问做显式引导（降低回复门槛）
- 精选留言的回复 = 二次内容，拉长停留时间

**检查清单**：
- [ ] 正文中有 1 个开放式问题 或 争议性观点
- [ ] 结尾有明确的互动引导语（非"欢迎点赞关注"）
- [ ] 互动设计让读者觉得"我有话想说"

---

### D6 · 关注转化（吸粉率 1-3%）

**核心公式**：不硬塞二维码，用系列感 + 资源钩子 + 人设差异化

- 系列感："这是第 X 篇，上篇讲了..."
- 资源钩子："回复 XX 领取..."
- 人设差异化：让用户感知关注后有持续价值
- 目标：形成"不关注就亏了"的心理

**检查清单**：
- [ ] 有系列感 或 资源钩子 或 人设差异化 中的至少一个
- [ ] 引导关注是自然的价值延伸，不是硬广
- [ ] 读者能感知"关注这个账号有什么好处"

---

### D7 · 传播裂变（二次传播占比 ≥ 60%）

**核心公式**：至少一个"可截图传播的信息单元"

- 一句金句 / 一张数据图 / 一个对比表格
- 让用户不用转发全文，也能在群聊中"随手甩一张图"
- 可截图单元要能脱离上下文独立传播

**检查清单**：
- [ ] 至少有 1 个可截图传播的信息单元
- [ ] 该单元能脱离上下文独立表达观点
- [ ] 金句精炼，有节奏感，能独立传播

---
